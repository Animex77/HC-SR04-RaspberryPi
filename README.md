# HC-SR04-RaspberryPi

### Dependences:

WiringPi library is needed for this library. See:
https://github.com/WiringPi/WiringPi

### Installation:
    
```bash
git clone https://github.com/Animex77/HC-SR04-RaspberryPi.git
cd HC-SR04-RaspberryPi
mkdir build && cd build
cmake ..
make
sudo make install
```
